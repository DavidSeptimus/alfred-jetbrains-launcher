# Alfred workflow

The Alfred frontend for the JetBrains project launcher — the keyword/IDE table
(`workflow/ides.json`) and the `info.plist` generator (`cmd/genplist`). It drives
the shared `jb` binary built from the repository root.

See the [repository README](../README.md) for usage, configuration, installation,
and how the workflow is built (`make` targets live at the repo root).
